Graphics.h Installation

Quick and easy installation guide for the graphics.h library on Ubuntu systems. Set up your graphics development environment with a single command.
Copy and paste the following command in your terminal.
Installation time: 5-15 minutes
sudo apt update && sudo apt install git -y && \ git clone https://github.com/AlbatrossC/graphics.h.git && \ cd graphics.h && \ chmod +x install.sh && \ ./install.sh
